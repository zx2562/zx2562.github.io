直伤爆发魂兽师

  2020.04.13

概述

  • 直伤爆发魂兽师 是一套集 高爆发 和 高机动性 于一体的直伤 PvP 配置。

特性 与 技能

特性

技能

宠物（可选）：

  • 幼龄巨貘 （ 拱地觅探 ）：通过 拱地觅探 有几率获得 等离子体（ 吸收等离子体 ），除此之外还有几率找到其他消耗品：一撮毛发（ 目盲碎毛 ）、羽毛（投掷羽毛，效果与 目盲碎毛 相同）。
  • 幼龄岩石瞪羚 （ 甩头猛冲 ）：提供打断技能，本身也拥有优异的爆发输出能力。

你也可以将以上两个宠物之一替换为 幼龄雾鳞蜥 （ 障目烟云 ），它们在 野兽模式 合体后提供的属性完全相同：

  • 使用 幼龄雾鳞蜥 施放 障目烟云 ，随后与 幼龄雾鳞蜥 合体，在烟雾区域内施放 大地冲击 获得 3秒 潜行 。

治疗技能（可选）：

  • 巨魔药膏 ：与特性 荒野知识 和 大地之子 连同使用，可清除 4个症状。

配置说明

魂兽师 特性 和 技能

  • 魂兽师 允许宠物与本体合二为一，并在合体后可继承一部分来自宠物的属性。合体技能 野兽模式 有 10秒的 CD。

  • 在战斗中无法切换宠物，因此在进入战斗前务必切换好宠物：

    • 幼龄岩石瞪羚 是你在战斗时应使用的宠物。
    • 幼龄巨貘 是你的非战斗宠物，你只需要通过 拱地觅探 来获得一个消耗品，最好是找到等离子体（ 吸收等离子体 ）。在结束一场战斗后，切换至 幼龄巨貘 寻找消耗品，并在找到后尽快切换至另一个战斗宠物。
    • 野兽模式 和 离开野兽模式 会触发切换宠物相关的特性。

  • 与宠物相关的特性在合体后依然有效（例如 宠物威勇 ），但受益者将变成本体，尽管属性有时会略有降低。

  • 影响你宠物的技能，你必须合体后再使用（例如 "干掉他们！" ），已经生效的技能会在合体后消失。
  • 在合体后使用 "群攻之力！" 将使你的每次攻击都可获得 威能 （单次打击只能堆叠一层 威能 ）。 万箭齐发 的 AOE多目标伤害能力，可以使你快速堆叠至 25层 威能 ，并在随后使用 高速连射 （长弓2）、 重击 （巨剑2）、 大地冲击 （ 野兽模式 F3） 来制造爆炸输出。
  • 与宠物合体通常可以制造高额伤害，但有时也可以获得其他能力，例如与 幼龄岩石瞪羚 合体后获得 冲锋 来提供机动能力，或与 幼龄雾鳞蜥 合体后获得 烟雾突击 获得闪避效果。你也可以与阵亡的宠物合体，然后退出合体，以便立即复活它们。

伤害技能

  • 适时使用 重击 （巨剑2）获得 借机攻击 的效果，来提升下次攻击 25%的伤害，最好衔接 大地冲击 使用。
  • 最佳爆发连招相当简单：
  1. 在使用长弓技能前，确保你身上拥有 激怒
  2. 通过 和风疾速 或 和风疾行 获得 急速
  3. 进入 野兽模式 并获得额外属性
  4. "干掉他们！" 使自身获得增伤
  5. 近身平射 并触发特性 狂暴双体
  6. 高速连射 打出成吨伤害

常规

  • 在游戏开始前的倒计时阶段，使用 "同心治疗！" 触发 嘹亮回响 来为队友提供 迅捷 。
  • 闪避 将获得 保护 （ 同伴之御 ），获得 保护 也将提供治疗（ 粗糙生长 ）。
  • 生存技能可以通过触发 荒野知识 清除两个症状并获得 激怒 。生存治疗技能 巨魔药膏 还可以通过触发 大地之子 额外再清除两个症状。
  • 使用 猛扑 来获得位移，并增加机动性。
  • 和风疾速 还有其他用途：
    • 加快攻击速度，最显著的就是加快 高速连射 和 巨剑普攻的劈砍。
    • 通过切换宠物，清除自身两个症状。
    • 获得 奔驰疾速 帮助你逃脱敌人的追击，或在你手持巨剑时追击敌人。
