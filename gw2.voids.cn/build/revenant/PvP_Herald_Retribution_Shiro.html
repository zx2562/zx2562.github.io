直伤惩戒预告者

  2020.04.15

概述

  • 直伤惩戒预告者 是一套放弃使用法杖的 PvP 新玩法。

配装说明

武器（可选）：

  • 盾牌 替换 利斧：盾牌的抗压能力更好，并可通过 顽强执念 获得更强的防御能力。

法印（可选）：

  • 如果需要更好的对抗症状队伍，可以将 勇气法印 和 智慧法印 替换为 净化法印 和 洁净法印 。

配置说明

伤害技能

基本玩法：

  • 直伤刺客预告者 当前版本不再拥有以往较强的续航能力，在没有格挡或闪避时，长时间停留在近战范围内做输出，可能导致自己很快被击败。虽然常规上认为 预告者 是一个近战输出职业，但他有相当多的技能都可以从远处发挥重要作用。该配置的核心玩法是利用游击战术，对目标进行短暂的爆发性输出。
  • 使用诸如 元素轰击 、 冰冻隔离 、刺客姿态下的 真正本质 、 桎梏波动 从一个安全距离消耗敌人的血量，冒然进入近战缠斗很容易吃到大量的 AOE伤害。
  • 利用你的控制技能，例如 瞬现裂隙 或 混乱疾风 将敌方团队打开一个突破口，使用 急速 、 附魔匕首 、 压倒优势 进一步增加你的爆发输出，让每一刻都变得更有价值。
  • 桎梏波动 和 死亡打击 是这套配置中的主要输出技能。
  • 无情突击 在 1v1 中效果最好，因为其伤害不会被额外的目标分摊。在团队作战中，该技能主要被用于闪避攻击。用该技能攻击一些有 潜行 能力的敌人时，注意要先使用 黑暗多面体 → 黑暗凝视 为目标施加 现身 ，因为敌人成功 潜行 后会让该技能中断。
  • 位面穿行 非常适合在快速位移到近战距离发起战斗，在获得 急速 的同时，还能使接下来的两次攻击不可被格挡。施加的 急速 有助于施展爆发输出 或 是直接使用普攻将目标血量压低，而不可被格挡的特质，也将那些企图利用格挡做最后挣扎的敌人打入倒地状态。
  • 翠玉风暴 几乎不会被使用，它会直接耗光你的能量，尽管有机会将敌人控制住，但是后续你无法使用任何能量做出有效输出。仅在个别情况下使用，例如帮助队友创造击杀机会，或是中断敌人的复活 或 终结引导。
  • 回荡喷发 是钉锤最具杀伤力的 AOE技能，可从远处施放，尽量CD好了就用。
  • 寒流闪袭 是一个无法被格挡的突进技能，搭配 智慧法印 后，也具备一定的爆发潜力。通常情况下，在切换至 锤 / 斧 武器组时，第一时间就该使用它。

铺垫伤害：

  • 在发起爆发输出前，有很多伤害可以进行铺垫。
  • 附魔匕首 是一个生命虹吸技能，这种技能伤害的计算，会绕过任何伤害减免的措施，如 保护 甚至 痛苦忍耐 。尽管如此，它依然不能绕开 闪避 或 无敌（如 失真 ）。
  • 压倒优势 在激活状态下可显著增加任何伤害技能输出能力。但是它持续消耗的能量比较多，仅应在爆发输出的几秒内激活。注意， 压倒优势 只能追加对单目标的输出，但它可以通过堆叠 强力执念 来增加 AOE 输出技能本身的伤害。如果你的能量即将耗尽，在达到 0能量前主动关闭 压倒优势 只会让该技能进入 1秒的CD，若能量耗尽让其自行关闭，则会进入 4秒的 CD。
  • 利用 传奇刺客姿态 下的 真正本质 （ 自然多面体 第二段）触发 龙吟回响 让 自然多面体 的被动持续几秒。

远程消耗连招示例：

  1. 附魔匕首
  2. 位面穿行
  3. 压倒优势
  4. 桎梏波动 （单手剑4）
  5. 武器其他技能 或 切换至钉锤使用 5技能 瞬现裂隙

混战连招示例：

  1. 混乱疾风
  2. 可选 - 切换至 传奇刺客姿态 激活 压倒优势
  3. 桎梏波动 （单手剑4）
  4. 死亡打击 （单手剑5）
  5. 冰冻隔离 （单手剑2）

续航 及 清症

  • 圣光多面体 → 圣光灌注 可以在使用后将受到的伤害全部转化为治疗，并获得无敌。至少应从该技能中获得 10k的治疗。小心在开启后敌人对你放弃攻击，在开启 圣光多面体 后，延迟几秒再使用 圣光灌注 通常可以避免该情况发生。 圣光多面体 有读条，而 圣光灌注 是瞬发的，并且可以在昏迷时使用。
  • 激活 混沌多面体 可通过 抗性符文 获得 抗性 。
  • 基于 恒定回春 的层数每秒可获得治疗，因此 压倒优势 也具备一定的续航能力，从而使你在利用 压倒优势 逃离敌人的追杀时也能获得治疗。
  • 当你需要使用 附魔匕首 获得治疗时，最安全的办法就是在类似 无情突击 这样的的 格挡 或 闪避 技能施放过程中使用。

常规

  • 切换姿态可以将能量恢复至 50点，移除一个症状（ 净化引导 ），并触发各种特性获得增益（如： 唤醒折磨 、 灵魂增益 ），还能触发法印的切换武器效果（如果不处于CD中）。
  • 切换姿态是能量管控方法中重要的一环，尽量在能量见底或者主要技能都进入CD后进行切换。

  • 开局时切换至 传奇巨龙姿态 ，在基地中开启以下技能为队友叠加增益：

    • 元素多面体
    • 力量多面体
    • 黑暗多面体
    • 自然多面体 （在即将开门时激活它，特别是它可以延长队友施加的增益，例如：游侠的 荒野召唤 ）

  • 在战斗中，尽量不要将 传奇巨龙姿态 下的通用技能开启超过 3秒。激活 元素多面体 和 力量多面体 ，然后立即使用 元素轰击 和 力量爆发 来制造可观的 AOE伤害。

  • 混乱疾风 可以将已经倒地的敌人击飞，可以利用这点将他们从争夺点上推出去，这可以在占据争夺点的同时，延缓它们的生命恢复。
  • 两种姿态下的精英技能都有明显的施法动画，如果在读条时，敌人对你施加了 目盲 ，立即使用 “取出/收起武器” 快捷键取消施法。当然，你也可以利用这一假动作欺骗敌人施放 闪避 或 格挡技能。

  • 黑暗多面体 是该配置中最灵活的技能：

    • 黑暗凝视 是一个低消耗的解除昏迷技能，并且可以为敌人施加 现身
    • 黑暗凝视 可以施加 目盲 阻碍敌人施放强有力的技能，从而避免浪费 闪避 ，并且在终结倒地敌人时（战士 / 魂武 / 工程 / 游侠 / 唤灵），使他们 目盲 以避免终结引导被打断，而施加的 现身 也很容易阻止倒地的 幻术师 使用倒地技能进行隐身。
    • 如果 你在施放 混乱疾风 前被敌人施加了 目盲 ，可以在 混乱疾风 即将施放完毕前，使用 黑暗凝视 抵消 目盲 ，让 混乱疾风 可以被安全施放。

  • 在非战斗状态下，移速增加上限为 33%，此时 迅捷 和 奔驰疾速 将拥有相同的移速。在战斗中， 压倒优势 将提高你的移速，这使你更容易脱战。

  • 传奇刺客姿态 下的 真正本质 （ 自然多面体 第二段 ） 是一个优异的 AOE 伤害技能。

被克制：

  • 高压制力的症状职业
