直伤步枪神枪手

  2020.04.15

概述

  • 直伤步枪神枪手 是一套直伤爆发型的 PvP配置，它可以将潜行者的爆发潜力发挥到极致，短短数秒就可以将敌人打入倒地状态。

特性 与 技能

特性

技能

通用技能（可选）：

如果你不需要 敏捷纹章 的症状清除能力，可以使用以下两个技能之一进行替换：

  • 束缚暗影 ：更好的爆发技能，使用前要先铺垫 神枪手标记 。
  • 刺客纹章 ：强力的爆发增伤技能。

配置说明

步枪 技能

  • 这套是主武器组，主要用于提供爆发输出、机动能力、潜行 以及一些通用性效果。
  • 认真的来说步枪拥有 10个不同的技能，但大部分都不会被使用。
  • 基本的玩法如下：使用普攻堆叠恶意 → 达到7层恶意后，进入 潜行 → 潜形后衔接 1技能 死亡审判 。
  • 屈膝 可以将其他 4个步枪技能强化，但会让你待在原地不能走动（使用 自由行动 即可起身）。在下蹲后，你依然可以使用 闪避 、传送、 紧急撤离 而不打破 屈膝 状态。使用 屈膝 是一种过于贪婪的游戏玩法，大多数情况下你不需要奢侈的使用这些技能，因此，尽量使用 单兵射击 进行远程输出。如果在实战中可以使用它，请利用以下技能堆叠恶意：
    • 三弹爆发 是最主要的伤害来源，并且可以堆叠 威能 和 恶意，尽量多使用它。
    • 观测射击 可以在紧急状况下为目标施加 定身 ，同时还能以更廉价的方式达到最大恶意层数（ 观测射击 花费 3点先攻值 与 三弹爆发 花费 6点先攻值获得的恶意层数相同），同时它还能提供 激怒 和 活力 。
  • 死亡撤退 是步枪武器组下类似 渗透暗箭 的技能，作为一个跳跃终结技还能与烟雾区域组合触发 潜行 。为了更好的施放该技能，建议将 “系统设置 - 常规选项 - 战斗/移动 - 自动选择目标” 的对勾取消掉。
  • 狙击手掩体 可以用来对付敌对的远程职业，但它最常见的用途还是触发组合技获取 潜行 ： 屈膝 → 放置 狙击手掩体 → 自由行动 → 在烟雾区域内使用 死亡撤退 获得 潜行 。
  • 在达到 7层恶意后，你应该进入 潜行 并使用 死亡审判 ，优先使用偷窃的技能触发 潜行 。除非目标的生命值已经很低，在这种情况下使用其他技能补足伤害即可。
  • 记得利用 能量法印 与 沉默隐望 协同使用。
  • 步枪爆发连招示例：
  1. 神枪手标记 并通过 恶意企图 将恶意堆叠至 1/7
  2. 单兵射击 × 3
  3. 使用偷来的技能 或 闪避 进入 潜行
  4. 死亡审判

剑匕 技能

  • 这套是副武器组，主要用于终结敌人（尤其在敌对阵容可以 反射或摧毁投射物 时）、在混战中自保。
  • 侧翼打击 → 侧翼突袭 是这套武器组下最主要的伤害来源，不要连续使用这个组合三次，因为这个连招的动作可预测性极高，滥用往往是在浪费先攻值。当然也有例外，在 侧翼打击 没有击中目标的情况下，你可以连续使用该技能获得闪避，以此来为被集火又无法逃脱的状况争取时间。
  • 渗透暗袭 是一个很好的开启战斗的技能，或者你可以利用它拉近与敌人间的距离。建议从视野死角的位置使用该技能（可以为你提供一条逃生路线），例如墙 或 柱子后面，这样你可以在被敌人集火，使用 暗影回归 回到原先位置，并清除一个症状。
  • 匕首之舞 通常作为一个远程消耗技能使用，仅在无法切换至步枪，又必须避免直接进入近战范围时才使用。也可以利用它的远程优势，破掉敌人的 圣盾 ，或是移除自身的 目盲 ，利用弹射效果清除 幻术师 的幻象也是不错的选择。
  • 隐秘突袭 尽管本身功能性强大，但却很少被使用。作为该配置中唯一的 潜行 来源，它可以将单手剑 1技能转变为 战术打击 ，从而打断敌人的施法（如 治疗 、终结 等）。当 横扫 未冷却完毕时， 隐秘突袭 提供的 潜行 将有利于你摆脱敌人的仇恨，有时甚至敌人会完全放弃追赶你。它高额的先攻值消耗 以及较长的读条时间，使你更需要考量施放的时机。

通用技能

  • 怜悯 可以刷新 神枪手标记 ，当你需要先攻值、需要切换标记目标 或者 需要触发 手法娴熟 时使用它（利用 慷慨无私 剥离并夺取敌人的增益（尤其是 稳固 ），并在随后施加 晕眩 以此来打断施法，如打断治疗技能、精英技能等）。

暗影疾步 是最容易在不同场合下发挥不同效果的技能，它的作用包括：

  • 为你的步枪变相提升 1200的攻击射程
  • 从混战中逃脱，当敌人追上你时，再使用 暗影回归 摆脱他们
  • 安全的终结敌人：开始终结敌人 → 暗影疾步 传送到安全位置 → 在终结快读条完毕时使用 暗影回归 回到敌人身边，以此来避免终结吟唱被打断
  • 解除昏迷 和 清症
  • 敏捷纹章 是你最重要的症状清除技能，它可以立即清除3个症状，注意这个清除效果对盟友有效，所以尽可能抱团使用。除此之外，主动使用它还可以补满你的耐力值，而这些都可以通过 闪避 变相转化为 潜行 。
  • 匕首风暴 是一个集进攻与防守于一体的精英技能。当你被地方集火时，长时间的闪避 及 加速移动的能力，使你很容易逃出生天。另外，该技能还能制造出非常可观的 AOE伤害，在避免进入危险的情况下便可对敌方队伍施压。需要注意的是， 电击光环 和 守护界线 可以打断该技能的持续施法，因此你依然不能大意。

神枪手相关技能与特性

  • 顺手牵羊 被 神枪手标记 所取代，并继承所有 顺手牵羊 可触发的特性，它的可施法距离更远，但有吟唱时间。
  • 使用 神枪手标记 成功标记目标，可以叠加一层恶意。
  • 沉默隐望 为持有步枪提供一个获得 潜行 的方法，但是耐力条会限制它的使用。
  • 神枪手标记 根据目标职业的不同，偷来的技能也不相同，这些偷来的技能大部分被用于打伤害或获得 潜行 ，因此你可以比较随意的使用它们。值得注意的是，偷来的技能只能对标记目标使用，这在对抗进入 潜行 状态，并且血量较低的敌人时非常好用。
  • 神枪手 特有的职业资源叫恶意，主要通过攻击（暴击） 神枪手标记 的敌人来进行堆叠，本配置最多可堆叠 7层恶意（ 七级邪恶 ），每层增加 3%的伤害。潜行普攻，如 死亡审判 会消耗掉当前所有恶意，并根据消耗的恶意层数，进一步提升伤害。
  • 针对不同职业， 神枪手标记 可偷取的技能如下：
职业 偷取技能 说明
守护者 汲取防御 获得圣盾，让标记的敌人中毒。
魂武者 汲取抗性 获得抗性，对标记的敌人施加折磨。
战士 汲取力量 获得威能，让标记的敌人虚弱。
工程师 汲取耐久 获得保护，对标记的敌人施加易伤。
游侠 汲取机动 获得奔驰疾速，定身标记的敌人。
潜行者 汲取精准 获得激怒，让标记的敌人目盲。
元素使 汲取暖意 获得活力，冰冻标记的敌人。
幻术师 汲取时间 获得急速，让标记的目标迟缓。
唤灵师 汲取生命值 治疗自己，让标记的敌人流血。
  • 达到或超过 5层恶意后，这些偷取的技能还将提供 3秒 潜行 ，但是要小心，这些技能也会造成伤害。如果你已经处于 潜行 状态下，直接使用它们会导致你进入 现身 状态，有一个办法可以避免该情况发生，让标记的目标处于自身的视野盲区后，再使用这些技能，就能让你进入 潜行 并避免 现身 。
