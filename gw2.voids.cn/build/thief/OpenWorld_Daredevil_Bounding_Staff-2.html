直伤闪避独行侠

  2020.06.17

概述

  • 直伤闪避独行侠 上手简单，高频率的攻击回血让初入游戏的小白们多了几分容错。

特性 与 技能

  • 诡诈
  • 武艺

特性

  • 越战越勇 ：如果不需要 重生精准 提供的小额治疗，可以点出增加伤害。
  • 顽强斗士 ：如果你需要携带 活力引导 、 强横防御 或更多体术技能。
  • 避祸脱逃 ：需要更多清症能力，可以点出。
  • 神行无阻 ：将闪避转变为冲刺技能，并获得伤害减免。

技能

治疗技能（可选）：

  • 恶意纹章 ：如果你受到的瞬间伤害并不高，可以利用该技能持续性获得小额恢复。
  • 遁入暗影 ：用于治愈并清除 燃烧 、 中毒 、 流血 、 困惑 。

通用技能（可选）：

  • 刺客纹章 ：被动提升伤害能力。
  • 战斗冲动 ：解除昏迷，并提供进攻型增益。
  • 暗影疾步 ：可用于解除昏迷 或 清除症状，是一个机动能力很强的技能。
  • 障眼烟幕 ：使敌人 目盲 并格挡投射物。
  • 绊索陷阱 ：可选的控制型技能。
  • 暗影庇护 ：获得 潜行 ，可以利用隐身效果与周围有精英怪的 英雄点 或 专精点 交流，从而避免鏖战。

精英技能（可选）：

  • 同仇敌忾 ：伤害能力更为出色。
  • 石蜥毒液 ：一个强力的控制技能，附近有盟友时，施放该技能后他们也可获得技能效果。

特性说明

武艺 特性分支可以使你的本体伤害提升更多，但放弃 诡诈 分支会让你牺牲一部分获取 激怒 和 活力 的能力。

配置说明

技能使用

伤害技能：

  • 这套配置的输出能力很强，仅依靠 虚弱冲锋 、 铁拳乱舞 和 普攻就可以打出非常高的伤害。
  • 不断的使用 飞天杖舞 可以打出高额的AOE范围伤害。
  • 使用法杖技能时，会通过 法杖大师 不断获得耐力，当耐力快满时，就使用 闪避 触发 弹跳闪避 来制造伤害。
  • 使用 顺手牵羊 获得额外的增益（ 致命快感 ），空放该技能也会获得效果。

防御技能：

  • 这套配置中，你可以使用非常多的闪避类技能，如 衰弱弧击 、 飞天杖舞 。
  • 如果交战中你迫切的需要耐力，请使用 敏捷纹章 。
  • 使用 强横防御 格挡攻击。
  • 冲撞打击 可以中断敌人危险的攻击。
  • 障眼烟幕 可以用来 目盲 敌人 或 格挡投射物。
  • 匕首风暴 可以在制造伤害的同时，反射投射物很长时间。

说明

续航能力：

  • 如果你觉得自己太脆，血量经常无法保持健康状态，可以点出特性 重生精准 。
  • 活力引导 仅有 16秒的CD（依赖特性 顽强斗士 ）。
  • 在危机的情况下可以使用 敏捷纹章 清除症状，如果需要更强的清症能力，请点出 避祸脱逃 或携带 遁入暗影 。

潜行：

  • 潜行 是 潜行者 非常有用的一项特殊能力，使用 暗影庇护 获得 潜行 可以更方便的与 英雄点 或 专精点 进行交流。
  • 如果 暗影庇护 的隐身时间依然不够用，可以装备短弓，对 障眼烟幕 的烟雾区域施放 集束炸弹 ，通过触发组合技获得长时间的 潜行 。

控制技能（破蔑视）：

  • 冲撞打击 和 浮空勾拳
  • 石蜥毒液
  • 强横防御 → 反身打击
  • 铁拳乱舞 → 铁掌攻心
  • 爆头射击
