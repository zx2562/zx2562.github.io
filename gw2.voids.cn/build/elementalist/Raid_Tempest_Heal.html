治疗暴风使

  暴君之刃
  2019.12.24

概述

  • 治疗暴风使 是一套极其灵活的辅助型配置，它可以提供持续型治疗 和 爆发型治疗，同时还可以提供多种增益。
  • 治疗暴风使 可以作为团队中继 治疗德鲁伊 之后的第二个治疗配置使用。
  • 使用 匕号 可以利用 热力同步 为小队成员传播增益，亦可使用 砂石狂啸 为整队成员延长增益持续时间，并提供 保护 。
  • 治疗暴风使 需要一定的实战经验，略微修改配置以适应小队的各种需求，推荐开荒 或 路人队伍使用。

  • 精力暗涌 ：不需要解除昏迷可以选用该特性为队友提供 活力 。

治疗 和 清症

  • 切换到 流水协调 时，尽可能贴近 boss 使用 流水爆弹 。
  • 在必要时，可以使用战吼类技能来获得一些额外的治疗，以及 再生 和 活力 。
  • 使用 间歇温泉 来进行远距离的治疗。注意，点出特性 秘法复活 （秘法）后，该技能可以用来为倒地队友加血，如果团队中有人即将倒地（血量极少），应看情况保留该技能。
  • 使用 治疗之雨 来进行小额群体性治疗（用于治愈团队损失的小额生命值），使用 流水过载 来进行巨幅群体性治疗。并且它们在吟唱时，将持续清除队友身上的症状。
  • 间歇温泉 和 治疗之雨 施放后可以提供 流水区域，与 爆炸类技能（如 寒冰尖刺 、 "天崩地裂！" ）触发组合技，可以提供群体治疗。
  • "抚慰伤痛！" 有三段治疗效果，第一段范围较小但治疗量最高，后两段范围逐渐增大但治疗量较低，在吟唱完毕后，还能移除附近队友一个症状。
  • 在点出 复原翻滚 的情况下，可以使用闪避清除队友身上的冰冻及燃烧效果。

主要增益提供

说明：

  • 点出特性 暴风咏叹 （暴风使）时，战吼技能可以将作用目标扩大至10名玩家。
  • 使用 大地+流水 或 空气+流水 特性分支时，注意要将 流水 分支下的 强效光环 点出。
  • 以下统计出的持续时间需要装备及消耗品提供 100% 的增益持续时间。

威能 ：

  • 火焰过载 ：在读条过程中它就能源源不断的为队友提供 威能 ，施法完毕后会在地上留下一片火焰区域，并为进入其中的队友继续提供 威能 。最终可提供 10层 威能 （32秒）。
  • "炽火燎原！" ：5层（本身3层 + 暴风咏叹 2层）（20秒）。
  • 暴风咏叹 （暴风使）：每个战吼技能可提供 2层（20秒）。
  • 火焰区域（如 熔岩喷泉 、 火焰过载 ）+ 爆炸终结技（如 "天崩地裂！" 、 寒冰尖刺 ），每次触发可提供 3层（40秒）。
  • 在 流水协调 时先施放 寒冰尖刺 ，在冰刺未落地时，立即切换至 火焰协调 并施放 熔岩喷泉 可以触发组合技叠加 3层 威能 （40秒）。

激怒 ：

  • 和风增益 （空气）：所有可提供光环的技能，都可用于叠加 激怒 （10秒） 和 迅捷 （10秒）。

保护 ：

  • 大地过载 ：在读条过程中它就能源源不断的为队友提供 保护 ，施法完毕后会在地上留下一片区域，并为进入其中的队友继续提供 保护 。
  • 元素护盾 （大地）：所有可提供光环的技能，都可用于叠加 保护 （6秒）。

再生 ：

  • 治疗之雨 、 流水过载 （技能施展完毕时提供）、 "极寒冰结！"

说明

光环：

  • 和风增益 （空气）和 元素护盾 （大地）可以依靠光环为队友提供 激怒 、 迅捷 和 保护
  • 由于特性 强效光环 （流水），使得每一个自己获得的光环也能给予附近的队友，这不仅仅局限于技能提供的光环，通过组合技获得的光环也能给予附近队友，例如：使用 寒冰尖刺 作用于 寒冰禁地 可以获得 冰霜光环 并分享给附近队友。注意，倒地的队友无法获得光环。
  • 该配置的光环获得方式：
    • 电击光环 主要由 空气过载 产生。
    • 冰霜光环 主要由 流水过载 、 元素棱堡 、 抚伤寒冰 、 "极寒冰结！" 产生。
    • 火焰护盾 主要由 火焰过载 、 "炽火燎原！" 产生
    • 磁力光环 主要由 大地过载 、 "天崩地裂！" 产生。
    • "五行归元！" 可以在对应协调下产生相应的光环。

控制（破蔑视）：

  • 切换到 空气协调 ，施放 疾风爆发 。
  • 切换到 大地协调 ，在 BOSS 的移动路线上施放 大地动荡 和 地能波动 。如果不能直接切换回 流水协调 ，还可以继续使用 大地过载 。
  • 寒冰禁地 （法杖4）可以提供一部分破蔑视能力，但注意不要放置在 流水区域 上，否则会影响组合技优先级顺序。
  • 使用 "天崩地裂！" 时，请留意与流水区域触发组合技 。
