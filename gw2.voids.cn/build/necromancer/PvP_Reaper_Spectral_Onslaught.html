直伤急速夺魂者

  2020.04.11

概述

  • 直伤急速夺魂者 是一套围绕近战输出的直伤夺魂者配置，通过鬼灵技能来强化生命之力的恢复，从而在战斗中收割敌人。

特性 与 技能

  • 鲜血
  • 怨恨

特性

  • 现世妄执 ：提升防御及续航，尤其在团队有辅助的情况下，契合度更高。
  • 死亡占卜 ：在使用 战吼治疗技能 或 战吼通用技能 时，可以点出。

技能

治疗技能（可选）：

  • 症状吞噬 ：如果症状压力过大，可以考虑使用。
  • 鲜血法池 ：范围型治疗技能，与特性 生命仪式 配合可以复活队友。
  • "灵魂夺取！" ：如果症状压力不大，或者团队有辅助时，可以选用。携带该技能时，切记要点出 死亡占卜 特性。

通用技能（可选）：

  • 血肉地虫 （ 暗蚀传越 ）：机动型技能，让你更好的风筝对手。
  • "迷雾归来！" ：抵御直伤伤害。
  • "无尽苦痛！" ：反制型清症技能。
  • 鬼灵缚爪 ：硬控技能，将敌人聚集起来，并附带恢复生命之力效果。
  • 增益腐蚀 ：将敌人增益腐化，同时也是一个防御技能。
  • 威力法池 ：将进入法池内队友的症状转化为增益，并提供 威能 和 稳固 。使用法池时建议点出特性 吸血仪式 。

精英技能（可选）：

  • 巫妖形态 ：在正确的时机进入巫妖形态，可以快速歼灭对手，但也容易被对手克制，尤其在对方拥有摧毁或反射投射物时，将很难发挥作用。

配装说明

项链（可选）：

  • 劫掠者项链 ：提升体力。
  • 圣骑士项链 ：牺牲部分伤害换取更强的续航能力。

符文（可选）：

  • 速度符文 ：与特性 暗影疾速 （摄魂）搭配使用效果最佳，更适用于支援。
  • 抗性符文 ：对抗主症状输出的队伍，可以考虑使用。
  • 领导符文 ：如果你需要更持久的 急速 ，可以考虑使用。
  • 力量符文 ：激进的进攻型选择，若选用该符文，请务必在斧聚武器组安插 勇气法印 。

配置说明

夺魂之幕技能

  • 你的绝大多数伤害都来自 夺魂之幕 状态下打出，这些伤害能力将伴随 致命洞察 和 夺魂灭屠 而进一步增强。
  • 进入 夺魂之幕 起手施放 惊惧魔甲 是最常见的操作，它有助于你打满伤害而不受控制干扰，该技能本身也能降低受到的伤害，从而使 夺魂之幕 持续时间更长。该技能第二段 惊惧释放 仅应在你需要硬控 或 惊惧魔甲 效果即将结束时，才考虑使用。
  • 死亡冲刺 尽量穿过 处决镰刀 提供的寒冰区域，以触发组合技获得 冰霜光环 。
  • 灵魂漩涡 尽量用在被控制的敌人 或 倒地的敌人 身上（因为 中毒 还会减少倒地敌人受到的治疗 以及 敌对盟友的复活）。由于进入 夺魂之幕 就能获得 急速 （ 夺魂灭屠 ），使得该技能更容易被快速施放完毕。
  • 处决镰刀 是一个单体技能，最适用于低血量的敌人，可以先用 惊惧释放 恐惧 敌人后再使用 处决镰刀 ，以防止敌人 闪避 躲掉该技能。
  • 在其他 4个技能都进入 CD后，使用普攻输出。
  • 爆发小连招： 死亡冲刺 （很容易制造出高额伤害） → 惊惧释放 → 处决镰刀 → 灵魂漩涡 。

巨剑技能

  • 夜幕降临 是巨剑中最有价值的技能，其能力与 腐化法池 不相上下。 夜幕降临 在相当多的场合下适用，尤其在 1v1 的情况下（当然这并不是你的主要职责）。
  • 自动攻击动作太慢，几乎不会考虑使用普攻来输出。
  • 掘墓剑斩 与普攻一样有相同的问题，但至少它的伤害更高。当 夺魂之幕 处于 CD时，该技能通常会被用于斩杀敌人，并将其打入倒地状态。如果你觉得有机会，可以在常规对战中使用它，尤其在对抗坦克型配置的敌方玩家时。在退出 夺魂之幕 后残存的几秒 急速 （ 夺魂灭屠 ）使得 掘墓剑斩 更容易被快速读条完毕。
  • 死亡漩涡 主要用于堆叠 易伤 和恢复稍许生命之力。
  • 黑暗抓攫 是一个不错的硬控技能，但很难命中移动的目标。
  • 在施放完 死亡漩涡 、 夜幕降临 、 黑暗抓攫 后，你的大部分时间都会处于 夺魂之幕 状态下，并且可以在 夺魂之幕 中等待巨剑技能冷却完毕。

常规

  • 鬼灵技能 和 聚能器4技能 灵魂抓攫 都是获取生命之力的重要方式，尤其是 鬼灵铠甲 。

    • 尽可能在进入 夺魂之幕 前开启 鬼灵铠甲 ，这样在进入 夺魂之幕 后获得 保护 ，并且持续恢复生命之力也将有效延长 夺魂之幕 的持续时间。
    • 在生命之力未达上限时，尽可能使用 灵魂抓攫 获取生命之力。

  • 鬼灵影步 不仅可以用于 解控 和 恢复生命之力，该技能还可以有其他创造性的用途：

    • 激活 鬼灵影步 → 被敌人追击时从高处跳下去 → 当敌人跟随你跳下时使用 鬼灵逆行 回到高处。
    • 激活 鬼灵影步 → 快速跑出敌人的视野范围 → "严寒刺骨！" 读条 → "严寒刺骨！" 即将读条完毕时，利用 鬼灵逆行 快速回到敌人身边。该操作导致 "严寒刺骨！" 很难被敌人打断，尽管有一定的运气成分。
    • 激活 鬼灵影步 → 暗蚀传越 （如果携带 血肉地虫 ）→ 当敌人追上你时，再使用 鬼灵逆行 传送回去。该操作同样适用于终结敌人，终结引导不会被这两种传送打断。

  • 对于提高机动能力，可以使用以下方法：开启 夺魂之幕 通过 暗影疾速 获得 迅捷 → 死亡冲刺 → 死亡冲刺 结束前按下 处决镰刀 。 死亡冲刺 在抵达终点时会让角色停滞一小段时间，可以通过离开 夺魂之幕 来打破这一状态。

  • "严寒刺骨！" 有非常明显的动画效果，极易被敌人躲避，如果你感觉技能要放空，可以使用收起武器快捷键中断施法。当然你也可以故意使用这一动作来迷惑敌人（自行中断技能后，技能动画依然会被播放），以骗出敌人的关键性技能，而你仅需要付出使该技能进入 5秒 CD的代价。
  • 恐怖之爪 是该配置中出色的爆发技能。在你拥有 威能 、 急速 ，敌人拥有 易伤 时，尝试使用它制造更多输出。

以下列出了 增益 将会被腐化为的 症状：

  • 圣盾 ⇒ 燃烧 （1层，3秒）
  • 敏捷 ⇒ 冰冻 （3秒）
  • 激怒 ⇒ 目盲 （5秒）
  • 威能 ⇒ 虚弱 （5秒）
  • 保护 ⇒ 易伤 （3层，8秒）
  • 急速 ⇒ 迟缓 （3秒）
  • 再生 ⇒ 中毒 （6秒）
  • 抗性 ⇒ 定身 （2秒）
  • 反弹 ⇒ 困惑 （3层，5秒）
  • 稳固 ⇒ 恐惧 （1秒）
  • 迅捷 ⇒ 残废 （10秒）
  • 活力 ⇒ 流血 （2层，8秒）
