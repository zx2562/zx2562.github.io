直伤匕首破法者

  2020.04.06

概述

  • 直伤匕首破法者 是一套野外打怪配置，它伤害能力充足，擅长反击并使敌人失去攻击能力。

特性 与 技能

  • 力量
  • 武器（全程激怒）
  • 战术（小队作战）

特性

  • 强力正义 ，需要更多治疗手段，可以点出。
  • 碎裂射击 ，武器使用步枪，可以点出。

技能

治疗技能（可选）：

  • 伤势愈合 ：治疗能力中等，可用于清症。
  • "突破极限！" ：更强力的瞬时恢复技能，使用时可以填满怒气值。
  • 治疗纹章 ：如果短时间内不会受到大量伤害，可以选择使用。

通用技能（可选）：

  • 怒气纹章 ：如果你使用最佳配装，不需要依靠 "为了正义！" 提供 威能 。
  • 纪律旗帜 ：长时间保持在同一片区域战斗，其效果要优于 怒气纹章 。
  • 耐力纹章 ：被动提升耐力恢复速度，这将提高使用闪避的频率。
  • 公牛冲锋 ：机动及控制性技能，如果不需要解除昏迷，可以用来替换 践踏 。
  • 践踏 ：群体控制 并能 解除自身昏迷。
  • 牦牛纹章 ：被动型防御技能，如果短时间内受到大量伤害，可以用来替换 践踏 。
  • "重整旗鼓！" ：如果症状压力较大，可以用来替换 践踏 。
  • 痛苦忍耐 ：短时间内免疫直伤伤害。

精英技能（可选）：

  • 战斗旗帜 ：进攻型增益持续时间比 盛怒纹章 激活时持续更久，但冷却时间也更长。
  • 怒神附体 ：需要额外的控制技能，可以选择使用。
  • 弱化之风 ：需要移除增益，或需要摧毁投射物，可以使用。

特性说明

选择 武器 特性分支会损失一些输出能力，但相对而言你无需再管控 巅峰表现 的层数，并且可以将 飞踢 替换为其他技能。点出 优势打击 和 无路可逃 时，不需要 "为了正义！" 或 盛怒纹章 即可保持永久 激怒 。

特性

  • 碎裂射击 ，武器使用步枪，可以点出。

技能

特性说明

选择 战术 特性分支，可以通过 强化盟友 和 方阵力量 提升附近盟友的输出能力，同时你需要选用技能 力量旗帜 和 纪律旗帜 ，并点出 纪律 分支中的 双重战旗 。

特性

  • 碎裂射击 ，武器使用步枪，可以点出。

配置说明

技能使用

输出技能：

  • 你的主要输出技能是 旋风斧斩 、 双重打击 、 回旋斧斩 、 干扰戳刺 、 瓦思垂的毁灭 。在两组武器间切换并使用它们，其余时间使用 飞斧投掷 或 斧的自动攻击做填充，注意打满三次斧的普攻，第三段伤害最高。
  • 如果 利刃风暴 可以命中 4名以上的敌人，则使用它。
  • 盛怒纹章 和 "为了正义！" 可以提供多种增益，建议冷却即用。
  • 总是尝试利用 瓦思垂的毁灭 和 干扰戳刺 的增伤触发条件以制造更多输出。
  • 使用其他输出技能前，利用 飞踢 触发 巅峰表现 和 洞察袭击 来增伤。

防御技能：

  • 全力反击 是一个很强的防御型技能，它不同于闪避，只是将受到的攻击伤害降至 0，如果你为了完成成就，需要不被一些特定技能击中， 全力反击 将不能帮助你完成此类成就。
  • 干扰戳刺 可以用于打断敌人的施法。
  • 可以选用技能 耐力纹章 、 牦牛纹章 或 痛苦忍耐 减免伤害。
  • 利刃风暴 可以反射投射物， 弱化之风 可以摧毁投射物。
  • 尽可能使用 闪避 躲避敌人的攻击，除非你能确保使用 全力反击 豁免敌人的攻击。

说明

续航能力：

  • 你可以通过 日月同辉 获得持续性的小额治疗，如果需要更多，还可以点出 强力正义 。
  • 切换武器 以及 使用 伤势愈合 都可以治愈症状，面临症状更多的战斗，可以使用 "重整旗鼓！" 。

控制技能（破蔑视）：

  • 干扰戳刺
  • 碎颅
  • 锤柄重击
  • 大地震颤
  • 飞踢
